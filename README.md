# go-couchdb-query-engine
CouchDB query implementation in Golang for in-memory object querying

## Features
- skip
- limit
- selector

## Selector
The selector query accepts following couchdb operators:
- all
- and
- elematch
- eq
- exists
- gt
- gte
- in
- lt
- lte
- mod
- ne
- nin
- nor
- not
- or
- regex
- size
- type
